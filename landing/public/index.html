// features

Designed for cluster power users

Every feature exists because we were frustrated with existing tools.

⊞
Multi-Panel Layout
Split the workspace horizontally or vertically. Drag panels out into separate OS windows, then bring them back. Your layout persists across sessions.
multi-panel splits
⚡
Real-time Watch
Powered by kube-rs list-watch streams. Resource state updates propagate instantly — no polling, no refresh buttons, no stale data.
kube-rs streaming
📋
Pod Log Streaming
Follow multiple Pod logs simultaneously in separate panels. Filter, search, and tail — all without leaving the app or juggling terminal windows.
concurrent log streams
🗂
Resource Management
List, inspect, and apply YAML for Pods, Deployments, Services, ConfigMaps, Secrets, and more. Full read-write access to your cluster resources from one place.
Pods · Deployments · Services
🦀
Lightweight & Fast
Tauri 2 wraps a Rust core. The binary weighs under 20 MB and starts instantly. No Electron, no Node.js runtime — just native system performance.
Tauri 2 + Rust
☸
kubeconfig Integration
Auto-discovers ~/.kube/config on launch. Switch clusters from the top bar. Works with any valid kubeconfig including OIDC and exec plugins.
automatic discovery
🖥
macOS Native
Ships as a native app for macOS Apple Silicon. No browser, no web server — behaves like a real desktop application.
macOS Apple Silicon
⌨
Pod Exec
Open an interactive terminal directly into any running Pod container. Built on xterm.js with full PTY support — no kubectl required.
xterm.js · PTY
🔀
Port Forward
Tunnel any Pod port to localhost in one click. Manage multiple active forwards simultaneously with start/stop controls.
local port tunneling
🔐
RBAC Aware
Checks your permissions via SelfSubjectAccessReview before showing destructive actions. Read-only users never see Apply or Delete buttons.
SelfSubjectAccessReview
🧩
CRD Support
Browse and manage Custom Resource Definitions alongside built-in resources. Inspect and edit any CRD instance with the YAML editor.
custom resources
